# SandraBedrukt - Gepersonaliseerde Cadeaus

Affiliate landingspagina voor gepersonaliseerde cadeaus onder €25.

## 📁 Bestanden

- `index.html` - Hoofdpagina
- `style.css` - Stijl
- `script.js` - Scripts
- `README.md` - Dit bestand

## 🚀 Deploy naar GitHub Pages

1. Push bestanden naar GitHub:
```bash
git init
git add .
git commit -m "Init"
git remote add origin https://github.com/paulnl/sandrabedrukt.git
git push -u origin main
```

2. Ga naar **Settings → Pages**
3. Kies **main** als brak
4. Opslaan

## 🔗 Affiliate Links Aanpassen

Zoek in `index.html` naar:
```html
https://partner.example.com/...
```

Vervang door eigen affiliate links van partners.

## ✏️ Teksten Aanpassen

Alle teksten en prijzen staan in `index.html`. Pas aan waar nodig.

## 📝 Structuur Uitbreiden

Voor nieuwe pagina's:
- Maak kopie van `index.html`
- Pas H1, productlijst en FAQ aan
- Upload als nieuw bestand
